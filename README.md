# Wordle
[![codecov](https://codecov.io/github/super-cooper/wordle/graph/badge.svg?token=YOTA60GQT4)](https://codecov.io/github/super-cooper/wordle)

This is a library for playing and automatically solving Wordle puzzles. It initially started as a
Python script, and then became a module of my website, written in Kotlin. It has since been
rewritten in Rust, and used as an opportunity to learn about the language and explore which patterns
I prefer for Rust software development.
